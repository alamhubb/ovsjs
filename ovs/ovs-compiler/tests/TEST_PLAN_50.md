## 测试策略

**优先级：**
1. P0（11-20）: 基础语法覆盖
2. P1（21-30）: 控制流完整性
3. P2（31-40）: ES6+特性支持
4. P3（41-50）: OVS核心特性

**测试方法：**
- 渐进式：从简单到复杂
- 单独运行：每个测试独立验证
- 透明进度：每完成一批（10个）记录一次

**验收标准：**
- 准确率：100%
- 无效映射：0
- 所有源码位置精确匹配生成代码
